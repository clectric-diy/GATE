# GATE
The "experiment by swapping" method is a cornerstone of Lunetta synthesis, treating logic chips like modular building blocks rather than fixed electronic components.

The idea is for this to be a Lunetta synth style module with 14 and 16 pin ZIF sockets to allow the subset of 4000 Series CMOS chips, which have compatible pinouts, to be swapped to change the sounds produced by the module.
